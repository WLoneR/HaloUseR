# HaloUseR

**This python written program uses the Auto-Code API to grab and display Halo Infinite user information.**

**Auto-Code API :** https://autocode.com/lib/halo/infinite/

**Instructions**

	1 - Run HaloUseR.
  	2 - Enter the gamer tag in the text box located at the bottom of the Form.
  	3 - Click 'Grab User Details'.
  	4 - Wait and receive juicy information.
